Web 浏览器

Google Chrome、Firefox 以及 Internet Explorer 是目前最流行的浏览器。

Google Chrome 浏览器

Google Chrome 浏览器是由 Google 开发的一款免费的开源 web 浏览器，于 2008 年发布。

Internet Explorer 浏览器

Internet Explorer 浏览器由微软开发的一款免费的 web 浏览器，发布于 1995 年。

Mozilla Firefox 浏览器

Firefox 浏览器是一款来自 Mozilla 的免费开源 web 浏览器，发布于 2004 年。
